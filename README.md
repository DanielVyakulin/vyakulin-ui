### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/vyakulin-ui.git
cd vyakulin-ui
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the development server:
```bash
npm run dev
# or

```

The application will be available at `http://localhost:3000`
